# fyne-clock
fyne ui clock demo
## Prerequisites
[fyne](https://developer.fyne.io/started/)

## Run
``
go run .
``

## Build
[packaging](https://developer.fyne.io/started/packaging)

``
fyne package -os windows -icon clock.png
``
